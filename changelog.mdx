<Update label="April 2026 — May 2026" description="v1.2.0">
  ### New Shipping Options
  <Badge color="green">Released</Badge>

  We've shipped a set of features to give you more control over your shipping rates.

  - **Transit Time Support** — Set delivery day ranges per rate using the new `Transit Time Min Days` and `Transit Time Max Days` columns. Values are validated as non-negative integers with `min ≤ max` before any change reaches Shopify; blank cells leave a rate's transit time unset.
  - **Free Shipping Thresholds** — Set a minimum order amount that triggers free shipping using the new `Free Min Order Amount` column (e.g., `100.00 GBP`). Above the threshold the rate is free; below it the normal price applies. Currency must match your store's primary market.
  - **Automatic Rate Grouping** — Rates sharing the same name and zone are automatically grouped together during import as a single Shopify shipping option with tiers. No new column to learn — grouping is implicit. Tiers must be contiguous (e.g., `0–49.99`, `50.00–99.99`), use the same condition type and currency, and any open-ended top tier must be last. The importer rejects mismatches with a per-row error so you can fix the spreadsheet before anything reaches Shopify.
  - **Better Validation Feedback** — Per-row errors now appear inline in the import preview, with one bullet per offending row, so you can fix multiple issues in one pass.

  <Note>
    These features are opt-in. Enable them in **Settings → New Shipping Options**. New columns are added to the downloadable example CSV when the toggle is on. With it off, your existing 13-column CSVs continue to work unchanged.
  </Note>
</Update>

<Update label="April 2026" description="v1.1.0">
  ### Live Snapshot Architecture
  We've completely rebuilt the app's backend to be faster, more reliable, and easier to use:

  - **No More Indexing** — MatrixRates now uses a "Live Snapshot" architecture. This means the app fetches your rates directly from Shopify in real-time, eliminating the need for background indexing and synchronization.
  - **Instant Exports** — Exporting your rates is now significantly faster and always reflects the most up-to-date information in your store.
  - **Simpler Workflow** — We've streamlined the import and export screens to make it easier to manage your shipping profiles and locations.
</Update>

<Update label="April 2025" description="v1.0.0">
  ### Initial Release
  The first version of MatrixRates Import/Export is now live on the Shopify App Store!

  - **Bulk Import & Export** — Manage hundreds of shipping rates at once using CSV files.
  - **Spreadsheet Editing** — Edit your rates in Excel, Google Sheets, or Numbers and upload them directly to Shopify.
  - **Multi-Zone Support** — Full support for complex shipping zones across different countries and provinces.
</Update>
